# Aurben Website

Aurben marketing site built with Vite, React, TypeScript, Tailwind CSS, and Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL that Vite prints in the terminal.

## Check Before Launch

```bash
npm run lint
npm run build
```

## GitHub Upload

Upload the project files to a new GitHub repository. Do not upload `node_modules` or `dist`; they are ignored by `.gitignore` and will be recreated.

## Vercel

Import the GitHub repository into Vercel and use:

- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

Calendly availability is controlled in the Calendly account, not in this codebase.
